# SSAA — Evaluación Física

Proyecto web para cálculo de pruebas físicas de la Escuela Antonio de Escaño.
Incluye login con Firebase, evaluaciones guardadas por usuario, exportación y gráficos.

## Características
- Registro e inicio de sesión (Firebase Auth)
- Guardado de evaluaciones por usuario (Firestore)
- Cálculo automático de notas según baremos PITO/PITA
- Mensajes personalizados por desempeño
- Exportación CSV y PDF
- Gráficos radar y barras
- Interfaz adaptada para GitHub Pages

## Tecnología usada
- HTML + CSS + JS
- Firebase (Auth + Firestore)
- Chart.js
- html2canvas / jsPDF
